박은성의 아케이드

브라우저 미니 게임 아케이드

클래식 아케이드부터 퍼즐, 전략, 카드, 단어 게임까지! 설치 없이 즐길 수 있는 미니 게임을 계속해서 확장 중입니다. 박은성이 직접 디자인하고 다듬은 총 18가지 미니 게임을 어떤 기기에서나 편하게 즐겨보세요.

컴퓨터 스마트폰 태블릿
캐주얼

지렁이 게임

방향키로 지렁이를 조작해 사과를 먹고 성장하세요. 속도가 점점 빨라지는 클래식 스네이크!
퍼즐

테트리스

떨어지는 블록을 회전하고 쌓아 라인을 지워 보세요. 레벨이 오를수록 속도가 빨라집니다.
두뇌

2048 퍼즐

같은 숫자 타일을 합쳐 2048에 도전하세요. 차분하지만 중독성 있는 퍼즐입니다.
샌드박스

모래 실험실

자유롭게 모래와 블록을 떨어뜨리고 쌓으며 물리 퍼즐을 해결해 보세요.
아케이드

세포 성장 게임

작은 입자를 흡수해 세포를 키우고 함정을 피해 생존하세요. 네뷸러스 스타일의 액션.
대전

레이저 핑퐁

마우스로 패들을 조작해 공을 반사하고 득점을 노려 보세요. 10점을 먼저 얻으면 승리!
아케이드

레트로 개구리

도로와 강을 건너 목적지에 도착하세요. 통나무를 이용하고, 차량을 피해 생존해야 합니다.
아케이드

네온 벽돌깨기

발판으로 공을 튕겨 벽돌을 모두 제거하세요. 콤보와 파워업으로 하이 스코어에 도전!
반사신경

플래피 비행

탭/스페이스로 새를 조작해 장애물을 통과하세요. 속도감 있는 점프 액션 게임입니다.
전략

지뢰찾기

논리와 추리로 지뢰의 위치를 찾아 플래그를 꽂아 보세요. 난이도를 선택할 수 있습니다.
기억력

카드 매칭

뒤집힌 카드를 두 장씩 선택해 짝을 맞추세요. 짧은 플레이로 집중력을 시험해 보세요.
퍼즐

소코반 창고지기

상자를 목표 지점으로 밀어 넣는 클래식 퍼즐. 되돌리기와 여러 레벨로 깊이 있게 즐겨보세요.
슈팅

우주 방위전

적 함대를 피하고 격추하며 보스를 방어하세요. 파워업을 모아 강력한 레이저를 발사!
러너

스카이 러너

장애물을 점프와 슬라이드로 피하며 더 멀리 달려 보세요. 속도는 계속 빨라집니다.
반사신경

두더지 팡팡

두더지가 머리를 내밀 때 빠르게 클릭해 점수를 얻으세요. 난이도별 폭탄과 콤보를 조심!
카드

네온 블랙잭

보유 칩을 관리하며 21에 도전하세요. 더블다운, 서렌더 등 핵심 전략을 구현했습니다.
단어

코드 워들

5글자 영어 단어를 6번 안에 추리하세요. 키보드와 색상 힌트로 논리력을 발휘해 보세요.
전략

하늘길 디펜스

펄스와 냉기 타워를 배치해 웨이브를 방어하세요. 에너지를 관리하며 최적의 방어선을 구축하세요.

즐기고 싶은 게임이 더 있다면 언제든지 제안해 주세요. 계속해서 새로운 아이디어를 실험하고 추가하겠습니다.

이 아케이드는 박은성이 기획·제작했습니다.
