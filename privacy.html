개인정보 처리 안내

개인정보 수집 기능 없음

GORICS.CLOUD NEWS는 공개 RSS 뉴스 데이터를 정적 파일로 보여주는 사이트입니다. 로그인, 회원가입, 댓글, 문의 폼, 광고 추적, 방문자 분석 스크립트를 사용하지 않습니다.

쿠키사용하지 않음
분석 도구Google Analytics 등 사용하지 않음
로그인/회원가입제공하지 않음
데이터 출처공개 RSS 피드

자동 갱신 방식

GitHub Actions가 매시간 공개 RSS 피드를 읽어 data/news.json 파일을 갱신합니다. 이 과정은 방문자의 이름, 이메일, 전화번호, 위치, 기기 식별자 등을 수집하지 않습니다.

RSS 텍스트 보호 처리

RSS 제목·요약에 이메일, 전화번호, 주민등록번호 형태, 카드번호 형태, 상세 주소 형태가 발견되면 게시 전 자동으로 [비공개] 처리합니다.

외부 링크

뉴스 카드를 누르면 원문 언론사 또는 뉴스 제공 페이지로 이동합니다. 외부 사이트의 개인정보 처리 방식은 해당 사이트 정책을 확인해야 합니다.
