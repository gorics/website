# os/linux/1 안내

작성일: 2026-03-29 (Asia/Seoul)

이 디렉터리는 브라우저에서 v86을 이용해 Linux 환경을 부팅하는 정적 페이지를 포함합니다.

## 현재 상태

- `index.html`은 같은 디렉터리의 `./linux.iso`를 직접 로드하도록 구성되어 있습니다.
- 따라서 이 디렉터리에 `linux.iso`가 존재하면, 공개 사이트를 통해 해당 ISO가 사실상 재배포됩니다.

## 중요한 주의사항

현재 저장소 기준으로는 `linux.iso`의 다음 정보가 문서화되어 있지 않습니다.

- 원본 출처
- 정확한 버전
- 원본 프로젝트명
- SHA256 등 해시값
- 포함 패키지 목록
- 재배포 시 필요한 라이선스/저작권 고지
- 소스코드 제공 의무 또는 제공 안내

즉, 현재 상태만으로는 `linux.iso`의 재배포 적법성이 검증되었다고 볼 수 없습니다.

## 이 문서의 의미

이 문서는 `linux.iso`의 라이선스가 확인되었다는 뜻이 아닙니다.
오히려 반대로, **현재는 출처와 라이선스 조건이 충분히 확인되지 않았으므로 주의가 필요하다**는 사실을 명시하기 위한 문서입니다.

## 권장 조치

1. `linux.iso`의 원본 출처와 정확한 버전을 확인합니다.
2. 원본 프로젝트의 라이선스 조건을 확인합니다.
3. 재배포에 필요한 고지 문서(`LICENSE`, `NOTICE`, 소스 제공 안내 등)를 함께 포함합니다.
4. 가능하면 외부 ISO를 그대로 재배포하지 말고, 공식 오픈소스 베이스에서 직접 재현 가능한 방식으로 이미지를 다시 빌드합니다.

## 검증 전 운영 원칙

위 항목이 확인되기 전까지는 다음과 같이 취급하는 것이 안전합니다.

- `linux.iso`를 "문제 없음" 상태로 홍보하지 않기
- 상업적/대규모 재배포에 사용하지 않기
- 출처 미확인 상태임을 명시하기

## 참고

이 문서는 기술적/운영적 주의사항을 기록한 것이며, 법률 자문이 아닙니다.
